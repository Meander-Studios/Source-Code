Toggle-able with slide animation, with click, and the toggle-able section is initiality not displayed

Click Me!
  • This is awesome, right!
  • Here's the Code: dropdownClick("Your Selector","Your Toggle-able Section",true,false);

Toggle-able without animation, with hover, and the toggle-able section is initiality displayed

Hover Me!
  • This is awesome, right!
  • Here's the Code: dropdownHover("Your Selector","Your Toggle-able Section",false,true);

Toggle-able with slide animation, with hover, and the toggle-able section is not initiality displayed

Hover Me!
  • This is awesome, right!
  • Here's the Code: dropdownHover("Your Selector","Your Toggle-able Section",true,false);

To implement it into your own code, simply link jquery and dropdownpro in a script tag, like this:

<script type='text/javascript' src='jquery.min.js'></script>
<script type='text/javascript' src='dropdowns.js'></script>

Here how the function works => if you want hoverable, use:

dropdownHover("Your Selector","Your Toggle-able Section",animation [true or false],initially displayed [true or false]);
